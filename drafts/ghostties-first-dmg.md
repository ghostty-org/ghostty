First DMG of Ghostties is out now. Free, open source, go grab it.

Ghostties is my fork of Ghostty. The pitch up front: multiple terminal sessions stacked in one window, vertical tabs down the side, built for running a bunch of agents at once. I'm usually 2 to 7 across projects in Claude Code, and I didn't want a mass of windows

The thing that still surprises me: I'm a designer, and I live in the terminal now (definitely not on my bingo card). Part of it is building Brukas, my native iOS and Mac app. There's no IDE I'm aware of with a simulator baked in like the web world has. You wire it up through Xcode and a desktop simulator, and the terminal ends up being where the work actually happens. Once I noticed that, I leaned all the way in.

Next up: task integration via MCP and CLI. I'm using Linear a lot more, so I'm building a way to pipe issues straight into Ghostties and spin up an agent per task in the sidebar.

It's early. Rough edges everywhere. But it's the thing I use all day.

Download: ghostties.org
Fork it, clone it, break it: github.com/SeanSmithDesign/ghostties

If you try it, tell me what you think, what's broken and what you wish it did. Trying to build in the open.
